Nuevas características están en desarrollo...

0.7.3

(10-09-2021)

  • Mapeo de mando en juegos, sin soporte para gamepads (Aviso: soporte de mando native en Genshin no funciona por el momento)
  • Solucionado: reinicio del ratón durante la habilidad definitiva en Genshin.
  • Movimiento del ratón ha sido suavizado.
  • Solucionado: escalado incorrecto en modo ventana. (editado)
  • Ahora es posible iniciar las aplicaciones directamente en PlayCover
  • Con la opción ‘Actualizar PlayTools’ en el menú puedes instalar los últimos cambios en las herramientas, sin necesidad de reinstalar la aplicación.
  • Si algo va mal, puedes eliminar la configuración de PlayTools con la opción ‘Reiniciar PlayTools’
  • La pestaña “Descargas” ahora tiene los links para las ultimas versiones desencriptadas de estas. (La lista se expandirá y hará mas dinámica en la siguiente versión)
  • Solucionado: error generado por la sincronización de documentos con iCloud / Dropbox
  • La barra lateral ahora tiene una anchura mínima y la aplicación un tamaño mínimo
  • La interfaz de usuario de descargas ha sido mejorada. Ahora mostrara un enlace de descarga basado en tu región.
  • La interfaz de configuración ha sido mejorada. Ahora puedes copiar el comando.
  • Soporte multilenguaje.
  • Solucionado: cierre inesperado al iniciar la aplicación para ciertos usuarios.
  • Ahora el botón Alt también oculta/muestra la barra de menú, facilitando el acceso a las configuraciones de ventana.
  • PlayCover lanza una notificación cuando termina de instalar una aplicación.
GitHub Release

0.6.2

(29-08-2021)

  • Elementos mapeados se re escalaran cuando el tamaño de la ventana cambie
  • Un indicador aparecerá cuando entres o salgas del modo de edición
  • Todos los UIViews escalan adecuadamente (Anuncios web dentro de Genshin Impact han sido arreglados, juegos como Genshin Impact aún necesitan reiniciarse para un escalado de ventana adecuado)
  • Solucionado: Barra de Menú que cubría la ventana de juego en modo ventana
  • Solucionado: El Mapeo de teclado no se guardaba para algunos usuarios
  • Solucionado: Genshin impact se cerraba en el menú de equipo para algunos usuarios
  • Mejorado: Gráficos mejorados x3 veces
  • Mejorado: Uso total de memoria reducido x2 veces
  • Solucionado: El movimiento de la cámara se reiniciaba después de una cinematica
  • La cámara ya no necesita manipulación adicional para funcionar correctamente luego del mapeo
  • PlayCover ahora tiene un sistema de reportes automáticos para cierres inesperados, el cual no solo funciona con la aplicación sino también con los juegos que hayan sido instalados con PlayCover
  • Se comenzó la migración de las configuraciones de juegos a PlayCover para un manejo comodo de estas y a su vez para sincronización

0.6.0

(22-08-2021)

  • Opción ‘Fix login’ esta disponible para todos los usuarios. Sigan las instrucciones del video para usarla, también encontraran una versión en texto aquí. Si tienen algún problema, por favor haga click en botón ‘Troubleshoot’. Ten en cuenta que todo esto es bajo tu propia responsabilidad, solo estoy proporcionando ayuda a los Supporters.
  • Mejorado: Desencriptación de aplicaciones ha mejorado significativamente desde la ultima versión.
  • Los elementos de mapeo ahora son guardados propiamente. No mas desaparición al azar.
  • Ahora el dock y el menú se ocultan durante el modo de juego, presiona Alt o la tecla opción para cambiar entre modos.
  • Solucionado: El ratón se quedaba trabado después de alguna cinemática en el juego.
  • Método de desencriptación alternativa ahora esta disponible para todos
  • LMB/ RMB / MMB ahora pueden ser asignados (utiliza Shift + R, L, M para asignarlos)
  • Reescalado automatico. Ahora la ventana se seescalara automáticamente en juegos como Wildrift, Among Us, etc.
  • Sin embargo, si quieres usar una resolución personalizada, utiliza Shift + Flecas para mover la pantalla de juego o Shift + ‘+’/’-‘ para cambiar el ancho o Control + Shift + ‘+’/’-‘
  • Usuarios con ratón externo. Actualmente necesitan desconectar y conectar el ratón después de cada configuración.
  • Planeado: característica para hacer opcionar la selección de Ratón/Trackpad en un futuro
  • Aviso para usuarios Intel
  • Solucionado: varios errores de la IU han sido solucionados.
