Petunjuk Keybindings

Terakhir diupdate untuk versi 0.7.0

    Tombol Keymapper

  • CTRL + P  →  Tombol mode Keymapper

    • Panduan Pengontrol:

  • CTRL + N Lalu Shift + A,X,B,Y
  • Shift + 1,2,3,4 for Dpad
  • Shift + 5,6,7,8 for L1,L2,R1,R2
  • Shift + 9 for Left Stick
  • Ctrl + M for Right Stick

    • Acara Tombol:

  • CTRL + J  →  Tambah W/A/S/D Joystick
  • CTRL + N  →  Tambah tombol baru
  • CTRL + M  →  Mouse/Gerakan Camera

    • Setelah Memilih Tombol Baru:

  • *key*  →  Tekan tombol apapun untuk di bind
  • SHIFT + L (L key on keyboard)  →  Bind Left Mouse Click
  • SHIFT + R (R key on keyboard)  →  Bind Right Mouse Click
  • CTRL + ‘+’  →  Increase Button Size
  • CTRL + ‘-’  →  Decrease Button Size
  • CTRL + DEL (Backspace)  →  Delete Button

    • Lain lain:

  • CTRL + F  →  Atur resolusi

    • Macros:

  • CTRL + U  →  Rekam
  • CTRL + I  →  Berhenti
  • CTRL + O  →  Replay Macro

    • Flow Control:

  • Press option  →  Tampilkan/Sembunyikan cursos
  • CTRL + W/A/S/D  →  Geser tombol

Apa itu PlayCover versi supporter?

Versi supporter adalah versi yang diberikan ke supporter sebelum dirilis ke pengguna umum. Versi ini memiliki fitur-fitur baru, tapi memiliki bug lebih banyak. Cek channel #how-to-donate di server discord kita untuk informasi lebih lanjut.

Kapan versi PlayCover selanjutnya akan rilis?

Kita mungkin tidak bisa memberikan tanggal tepat untuk rilis selanjutnya, tapi jangan khawatir karena kita bekerja keras untuk mengembangkan app dan merilis fitur baru!

Apakah SIP (System Integrity Protection) harus di disable? Apakah itu aman?

Sementara, iya. Tapi disable SIP tidak akan diperlukan pada rilis yang akan mendatang. Itu juga aman, terutama di Mac M1.

Bagaimana cara mengatur pengaturan Genshin?

Di dalam game, tekan tombol Option di keyboard untuk menampilkan cursor. Lalu tekan icon di kiri atas (sebelah map) dan klik settings. Disana semua pengaturan bisa diubah.

Gimana cara menambahkan klik kiri dan kanan?

Untuk menambahkan mouse click, pilih tombol dan tekan 'Shift + L atau R' (tombol L dan R di keyboard, bukan di mouse).

Gerakan kamera tidak berfungsi di game?

  • Tekan CTRL + M di keymapper lalu tempatkan di tengah.
  • Jika menggunakan mouse external, coba cabut dan pasang mouse.

Semua langkah yang ditampilkan di video sudah dijalankan dengan benar tapi keybindings masih tidak berfungsi.

Cek apabila game dipindah ke folder Applications setelah di install. Jika iya, hapus dan reinstall. Biarkan game berada di folder Documents.

PlayCover tidak berjalan meskipun sudah mengikuti petunjuk dengan benar (troubleshoot juga menunjukkan sistem sudah benar)

XCode harus di install dari App Store.

Apakah Genshin Impact harus reinstall tiap rilis PlayCover terbaru?

Iya, untuk menerapkan perubahan terbaru .ipa harus di install ulang. Tapi, Game Data tidak perlu di download, karena itu hanya akan menggantikan versi lama yang di patch.

Bagaimana cara reset command “sudo nvram boot-args…”?

Dengan menjalankan command berikut:


sudo nvram boot-args=""

Kenapa ada teks random muncul saat Mac di restart atau dinyalakan?

Namanya mode verbose, teks menunjukkan apa yang sedang dilakukan sistem. Verbose mode diaktifkan dengan command berikut:

sudo nvram boot-args...

Teks bisa di disable dengan menjalankan command berikut secara urut:
sudo nvram boot-args=""

sudo nvram boot-args="cs_enforcement_disable=1 amfi_get_out_of_my_way=1 keepsyms=1 intcoproc_unrestricted=1 amfi_unrestrict_task_for_pid=1 amfi_allow_any_signature=1 cs_debug=1"

SIP tidak bisa disable dan tidak bisa menambah boot-args karena *blank password*

Ini normal. Sistem nix memang mengosongkan password saat diketik. Ini hanya bentuk keamanan dan memang sengaja. Ketik password seperti biasanya dan tekan enter jika sudah.

Error “This application is not supported on this Mac” muncul.

Perangkat Intel sementara tidak didukung oleh PlayCover. Pantau terus karena developer sedang bekerja untuk mendukung perangkat Intel

Error “You do not have permission” muncul

  1. Pastikan SIP sudah disabled
  2. Atau masuk terminal dan ketik: (tanpa teks <>)
    chmod -R 755 <drag app here>
  3. Atau masuk terminal dan ketik: (tanpa teks <>)
    xattr -rd com.apple.quarantine <drag app here>

Keybindings di Genshin Impact bertingkah aneh dan restart app dan Mac tidak membantu

Coba:

1. Hapus dan reinstall app dengan memilih ‘clear app cache’.
2. Jalankan command berikut untuk menghapus file .plist yang mengandung pengaturan keybindings:

rm ~/Library/Containers/com.miHoYo.GenshinImpact/Data/Library/Preferences/com.miHoYo.GenshinImpact.plist
